# Nvis: Visualization of Temporal Interval Hierarchies

## Introduction

<div style="text-align: justify; font-size: 16px">
<span style="all: unset;">

<!-- Or use **`nvis`** -->
 <span style="font-family: monospace; font-weight: bold;">Nvis</span> is a visualization tool designed for exploring temporal interval hierarchies generated by the <span style="font-family: monospace; font-weight: bold;">nfer</span> tool. <span style="font-family: monospace; font-weight: bold;">Nfer</span> is a language and framework for event stream abstraction with several capabilities, one of which is deriving temporal intervals that satisfy certain temporal properties from event traces. These properties can be defined as *rules* in <span style="font-family: monospace; font-weight: bold;">nfer</span>, which can either be mined automatically or be defined manaully. You can access <span style="font-family: monospace; font-weight: bold;">nfer</span> in [this repository](https://bitbucket.org/seanmk/nfer/src/master/), and read more about the <span style="font-family: monospace; font-weight: bold;">nfer</span> formalism in the two following papers:

> Sean Kauffman, Klaus Havelund, and Rajeev Joshi. nfer -- a notation and system for inferring event stream abstractions. In International Conference on Runtime Verification, pages 235 - 250. Springer, 2016.

> Sean Kauffman, Klaus Havelund, Rajeev Joshi, and Sebastian Fischmeister. Inferring event stream abstractions. Formal Methods in System Design, pages 54 - 82. Springer, Aug 1, 2018.

<br></br>
